# b3-todo (mono)

# Structure
This mono repo contains multiple workspaces : **api**, **front** & **tests**

# Project instalation
```sh
# Clone the repo
git clone https://github.com/baptiste-jullion/b3-todo.git
```

```sh
# Install dependecies
bun i
```

# Run project
```sh
bun --filter '*' dev
```

# Acces app
Go to http://localhost:4200
