# b3-todo (mono)

# Structure
This mono repo contains multiple workspaces : **api**, **front** & **tests**

# Project instalation
```sh
# Clone the repo
git clone https://github.com/baptiste-jullion/b3-todo.git
```

# Run project
```sh
# for the first time
bun run setup
# already setup
bun dev
```

```sh
# run api build
bun api
```
