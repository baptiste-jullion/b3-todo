# b3-todo (mono)

# Structure
This mono repo contains multiple workspaces : **api**, **front** & **tests**

# Project instalation
```sh
# Clone the repo
git clone https://github.com/baptiste-jullion/b3-todo.git
```

```sh
# Install npm dependecies
npm i
```

# Run project
```sh
# front
cd front && npm run dev
```

```sh
# api
cd api && node .
```

```sh
# tests
cd tests && node .
```
