# b3-todo (mono)

# Structure
This mono repo contains multiple workspaces : **api**, **front** & **tests**

# Project instalation
```sh
# Clone the repo
git clone https://github.com/baptiste-jullion/b3-todo.git
```

```sh
# Install dependecies
bun i
```

# Run project
```sh
```sh
# front with bun
cd front/ && bun dev
```

```sh
# api
cd api/ && bun .
```
